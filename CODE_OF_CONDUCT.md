# 貢獻者行為準則

## 我們的承諾

為建立開放友好的社區，我們作為貢獻者和維護者承諾，無論年齡、體型、身體殘疾、民族、性別認同與表達、經驗水平、國籍、個人外表、種族、宗教或性認同及取向，為每位參與者提供無騷擾的體驗。

## 我們的標準

有助於創建積極環境的行為包括但不限於：

- 使用歡迎和包容的語言
- 尊重不同的觀點和經驗
- 接受建設性批評
- 關注社區最佳利益
- 對其他社區成員表現同情

參與者不應接受的行為包括但不限於：

- 性化語言或意象的使用及不受歡迎的性關注或性向騷擾
- 人身攻擊、侮辱性評論或仇恨言論
- 公開或私下騷擾
- 發佈他人私密信息（如物理或電子地址）而未得明確許可
- 其他可能合理認為在專業環境中不適當的行為

## 執行責任

社區領導者負責澄清和執行可接受行為標準，並採取適當且公平的糾正措施，以應對不可接受行為。

社區領導者有權和責任：

- 刪除、編輯或拒絕不符合本行為準則的評論、提交、代碼、維基編輯、問題和其他貢獻
- 暫停或永久禁止任何他們認為行為不適當、威脅、冒犯或有害的貢獻者

## 範圍

本行為準則適用於：

- 所有項目空間
- 代表本專案的公共空間（如社交媒體、發言人資格等）

## 執行

可以通過以下方式報告濫用、騷擾或不可接受的行為：

聯絡維護團隊：[maintainer@example.com]

所有投訴將被審查和調查。維護者有義務對報告人保密。

## 執行準則

社區領導者將遵循這些社區影響準則來確定他們認為違反本行為準則的任何行為的後果：

### 1. 更正

**社區影響**：使用不當語言或其他被社區認為不專業或不受歡迎的行為。

**後果**：私下書面警告，明確說明違規的性質。可能要求公開道歉。

### 2. 警告

**社區影響**：通過單一事件或一系列行為的違規。

**後果**：伴隨對持續行為後果的警告。在指定時期內不與相關人員互動，包括自發尋求與執行者的互動。這包括避免在社區空間和外部渠道中的互動。違反這些條款可能導致臨時禁令。

### 3. 臨時禁令

**社區影響**：嚴重違反社區標準，包括持續的不當行為。

**後果**：在指定時期內暫時禁止任何形式的社區互動或外部溝通。在此期間不允許與相關人員進行公開或私下互動。違反這些條款可能導致永久禁令。

### 4. 永久禁令

**社區影響**：展示持續違反社區標準的模式，包括持續的不當行為、對個人的騷擾或對群體的仇恨和歧視。

**後果**：無限期地永久禁止任何形式的社區互動。
